Darth Fader
===========

A feature rich fading carousel plugin for jQuery - Built Feb 2013

**User Guide**
------------

```javascript
$('#slider').darthFader({
		slideWrap : '#slide_wrap',
		controls : '#slider-controls',
		pagination : true,
		thumbPopUp : true,
		speed :  400,
		nextButton : '#next',
		prevButton : '#prev',
		auto: true
	});
```

**Features**
------------

- Optional pagination
- Optional pause on hover
- Sort randomly or as per DOM order
- Auto scroll option
